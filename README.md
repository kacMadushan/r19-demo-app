## Users List

This is demo users list web application. This application created for learn react update version (v19) key features and more.

### Tech Stack

    - React.js
    - TypeScript
    - Motion Js - (animations lib)
    - TailwindCSS

### Installation

Install the project dependencies using npm:

```bash
git clone https://github.com/kacMadushan/r19-demo-app.git
npm install
npm run dev
```

Open http://localhost:5173/ in your browser to view the project.
